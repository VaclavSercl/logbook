# TOOLS.md — Nástroje Njorora

## Co umím

### Git & GitHub
- `git` — všechny operace (commit, push, pull, branch, merge, rebase)
- `gh` — GitHub CLI pro repozitáře, PR, issues, CI
- Správa větví, code review, automatizace workflow

### Soubory & Kód
- Čtení, psaní, editace souborů
- Hledání v kódu (`grep`, `search_files`)
- Refactoring a optimalizace
- Code review a linting

### Terminál
- Spouštění příkazů na serveru "caslav" (RPi 4, Ubuntu 25.10)
- Instalace balíčků, správa procesů
- Build, test, deploy

### Web
- Vyhledávání a výzkum
- Interakce s webovými stránkami
- API volání (curl, Python requests)

### Plánování
- Vytváření plánů a tasků
- Delegace práce
- Sledování pokroku

## Jak pracuji

1. **Přímé akce** — soubory, kód, git → dělám sám
2. **Výzkum** — hledám informace, porovnávám možnosti
3. **Plánování** — rozdělím velký úkol na malé kroky
4. **Review** — zkontroluji výsledek před finálním schválením

## Konvence

| Věc | Pravidlo |
|-----|----------|
| Složky | vše lowercase |
| Commity | česky, popisně |
| Větve | `feature/nazev`, `fix/nazev` |
| Dokumentace | Markdown, česky |
| Kód | Python, čistý, PEP8 |
